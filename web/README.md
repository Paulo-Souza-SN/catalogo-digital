## Frontend (O que o usuário vê)
Para esta parte visual do projeto, decidi utilizar as ferramentas:
- React: A estrutura principal que faz o site ser interativo.

- Tailwind CSS: Utilizado para criar o estilo visual do catálogo de forma rápida e organizada.

- Lucide React: Para os ícones modernos que facilitam a compreensão visual.

- AOS (Animate On Scroll): Para adicionar animações suaves conforme o usuário rola a página.

- React Router: Responsável pela navegação entre a tela inicial e os detalhes dos produtos.

## 💡 Sobre a estrutura
O projeto utiliza uma arquitetura de componentes que separa a lógica de exibição da estilização. O uso do Tailwind permite uma interface "mobile-first", garantindo que o catálogo funcione perfeitamente em dispositivos móveis e desktops.
